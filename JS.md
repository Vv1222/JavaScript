## 数据类型

### 基本类型 7种

- string

	- 由'' "" 单引号或双引号包住
	- 有length属性
	- 可以用“+”号连接两个字符串

- number

	- js里面最开始只有一个数字类型，在内部被标示为64位浮点数，如1===1.0 // true

- boolean

	- null 、undefined、数字0、空字符串‘ ’ 、false、数字NaN 会被认为是假
	- true和字符串‘false’ 会被认为是真

- undefined
- null
- symbol
- bigint
